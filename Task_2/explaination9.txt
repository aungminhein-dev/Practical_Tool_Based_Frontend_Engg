Technologies : html, css, js

In this task, simple html page is carrying 3 button elements which possess uniques ids.

Based on their ids, they are marked as javascript variables :

code : 
    let redButton = document.getElementById("red");
    let greenButton = document.getElementById("green");
    let blueButton = document.getElementById("blue");

Then each button is wrapped with event listener click 
and change css property by javascript

code :
redButton.onclick = function () {
    document.body.style.backgroundColor = "red";
}
greenButton.onclick = function () {
    document.body.style.backgroundColor = "green";
}
blueButton.onclick = function () {
    document.body.style.backgroundColor = "blue";
}

Actually it is the manipulation of document object model (DOM) by using javascript.
